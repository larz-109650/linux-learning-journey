# linux-learning-journey

This repository documents my learning process in Linux, Git, Bash, 
Networking and Cybersecurity through technical documentation, 
practical labs and personal projects.

## Project objectives

- Develop a solid foundation in Linux.
- Master Git and GitHub through real-world projects.
- Learn Bash scripting.
- Understand networking from a practical perspective.
- Prepare for the Ethical Hacking course at UTN.
- Build a public technical portfolio.

## Methodology

Each topic is developed following a structured process:

1. Study of the theory.
2. Practical experimentation.
3. Documentation of results.
4. Version control using Git.
5. Review and continuous improvement.
